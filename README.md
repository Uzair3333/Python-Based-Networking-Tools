# Python Based Networking Tools

## Overview
This repository contains a collection of Python projects focused on networking and cybersecurity fundamentals. The projects are designed to help beginners and aspiring security engineers practice real-world networking concepts through hands-on coding.

Whether you are learning subnetting, log analysis, or preparing for red teaming tasks, these tools provide a solid foundation to build your skills.

---

## Projects Included

### 1. Subnet Calculator CLI
- Calculates network ID, broadcast address, first and last usable IPs
- Determines the subnet mask based on the number of required hosts
- Teaches subnetting, CIDR notation, and magic number calculation

### 2. Security Log Analyzer
- Reads log files and extracts IP addresses
- Counts the number of requests per IP
- Detects suspicious activity based on configurable thresholds
- Introduces log analysis and pattern detection for cybersecurity

## Getting Started
1. Clone the repository:
```bash
git clone https://github.com/YourUsername/Python-Based-Networking-Tools.git
```

2. Navigate to the project folder:
```bash
cd Python-Based-Networking-Tools
```

3. Run any project using Python 3:
```bash
python project_name.py
```

> Make sure to read the individual project README files for usage details.
